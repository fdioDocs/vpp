What is VPP?¶

The FD.io VPP platform is an extensible framework that provides out-of-the-box production quality switch/router functionality. The FD.io’s Vector Packet Processing (VPP) technology is a high performance, packet-processing stack that can run on commodity CPUs.

The benefits of this implementation of FD.io VPP are its high performance, proven technology, its modularity and flexibility, and rich feature set.
